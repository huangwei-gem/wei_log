# Wei Log — 个人作品集

> React + Vite 构建的暗色调设计作品集，支持展示图片和视频作品。

## 功能特性

- 🎨 暗色调视觉风格，红色强调色
- 🖼️ 响应式 Grid 作品展示
- 🎬 视频悬停自动播放
- 🔍 分类筛选（Brand / UI/UX / Motion / Art）
- ✨ 滚动淡入动画
- 📱 移动端适配

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 项目结构

```
wei_log/
├── src/
│   ├── App.jsx.js      # 主应用组件
│   ├── main.jsx        # 入口
│   └── style.css       # 样式
├── index.html
├── vite.config.js
├── package.json
└── README.md
```

## 自定义指南

| 替换内容 | 文件位置 |
|---------|---------|
| 你的名字 | `src/App.jsx.js` 中 Hero 组件的 `Your Name` |
| 个人照片 | Hero、About 区域的 placeholder |
| 作品内容 | `projects` 数组 |
| 联系方式 | Contact 组件 |
| 导航文字 | Navbar 组件 |

## 技术栈
