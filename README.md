Wei Log — 个人作品集
=========================================

React 暗色调设计作品集，支持展示图片和视频作品。
灵感来源于 caiovisuals.com

功能特性
--------
- 暗色调视觉风格，红色强调色
- 响应式 Grid 作品展示
- 视频悬停自动播放
- 分类筛选（Brand / UI/UX / Motion / Art）
- 滚动淡入动画

技术栈
------
- React 18（通过 CDN 加载）
- ES Modules（importmap）
- CSS3

本地预览
--------
直接用浏览器打开 index.html 即可。

部署
----
Cloudflare Pages 静态站点部署，无需构建。
